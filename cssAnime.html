Como inserir a animação

1. Salve o arquivo animacao.css em seu projeto;
2. Copie a classe que aparece no texto verde e insira na tag (HTML) que deseja fazer a animação

HTML Exemplo

<div class="anime-desaparecer"> </div >

CSS Exemplo

.anime-desaparecer { animation: 2s ease 0s 1 desaparecer } 
 
@keyframes desaparecer {
    0% {
        opacity: 1;
    }
    100% {
        opacity: 0;
    }
}

Editar o tempo, veja como funciona cada elemento

Passe o mouse por cima de cada elemento e veja sua função ou ver todas elementos

animation: 2s ease 0s 1 desaparecer animation animation: Propriedade - Contém todos elementos que manipulam a animação 2s 2s: Duração da animação - Quanto tempo, em segundos ou milisegundos, durará um ciclo da animação (o padrão é 0) 1 = 1 minuto. ease ease: Função de tempo da animação - Forma como a animação progride no tempo (o padrão é ease). Valores já pré-definidos que são: linear, ease, ease-in, ease-out e ease-in-out; 0s 0s: Tempo de atrazo da animação - Define a partir de quanto tempo a animação vai se iniciar (o padrão é 0) 1 1: Número de repetição - Determina o número de vezes que a animação vai se repetir (o padrão é 1). Podemos deixar a animação repetindo infinitamente, basta especificar o valor infinite. desaparecer desaparecer: Nome da animação - Essa função chama os css @keyframes que contém os tipos de animação
